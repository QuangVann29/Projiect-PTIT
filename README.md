# Website Khảo Sát Mức Độ Mất Tập Trung

## Giới thiệu

Đây là website khảo sát giúp người dùng nhận diện những yếu tố gây mất tập trung trong học tập, công việc và sinh hoạt hằng ngày. Sau khi hoàn thành khảo sát, hệ thống sẽ phân tích kết quả và đề xuất các giải pháp phù hợp nhằm cải thiện khả năng tập trung.

## Chức năng chính

* Khảo sát 5 nhóm nguyên nhân gây mất tập trung:

  * Công nghệ và mạng xã hội
  * Môi trường và tác động xã hội
  * Tâm lý và áp lực cuộc sống
  * Thói quen và lối sống cá nhân
  * Kỹ năng quản lý bản thân

* Hiển thị kết quả khảo sát dựa trên lựa chọn của người dùng.

* Đề xuất giải pháp phù hợp cho từng nhóm nguyên nhân.

* Tích hợp:

  * Hình ảnh minh họa
  * Video
  * Audio
  * Ảnh 360 độ (Momento360)

* Cho phép xem thêm các giải pháp liên quan.

## Công nghệ sử dụng

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Cloudinary (lưu trữ ảnh, video, audio)
* Momento360 (hiển thị ảnh 360°)

## Cách chạy dự án

1. Clone repository:

```bash
git clone <repository-url>
```

2. Mở thư mục dự án:

```bash
cd project
```

3. Chạy bằng Live Server hoặc mở file:

```text
src/page/home.html
```

## Kết quả khảo sát

Người dùng chọn 5 mức độ cho 5 nhóm nguyên nhân.

Sau khi hoàn thành khảo sát:

* Hệ thống xác định nguyên nhân chính.
* Hiển thị kết quả thống kê.
* Đề xuất giải pháp tương ứng.
* Cho phép xem thêm các giải pháp khác.
